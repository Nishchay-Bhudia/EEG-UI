Bluetooth Headband

Works with Muse, BrainBit, and any BLE EEG headband. Raw EEG is collected for 2 s then sent to your Render backend for full Random-Forest classification — or processed locally if no backend is configured. Requires Chrome or Edge.

Render Backend URL

The backend must expose POST /analyze (raw EEG) and optionally POST /analyze/bands (pre-computed powers).

Raw EEG endpoint

POST /analyze
{
  "eeg_data": [[ch0_s0, ch0_s1, ...], [ch1_s0, ...]],
  "sample_rate": 256,
  "ppg_ir":  [ir_s0, ir_s1, ...],   // optional
  "ppg_red": [r_s0, r_s1, ...]      // optional
}

Pre-computed bands endpoint

POST /analyze/bands
{ "delta": 0.12, "theta": 0.22, "alpha": 0.41,
  "beta": 0.18, "gamma": 0.07 }
